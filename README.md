# About India - Information Portal

A comprehensive portal providing information about India, its states, and culture. Built with React and Vite.

## Features
- Interactive state-wise information.
- Tourism highlights.
- Modern, responsive design.

## Tech Stack
- React
- Vite
- Framer Motion
- React Router

## How to Run Locally
1. Clone the repository.
2. Run `npm install`.
3. Run `npm run dev`.
